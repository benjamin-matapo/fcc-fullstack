<!-- Duplicate this block for each submodule -->

#### 💡 What I learned
<!-- Bullet the main concepts. Write them as if explaining to someone who has never seen this before. -->

- Main HTML elements like 
    - `<html>`
    - `<head>`
    - `<body>`
    
    and attributes

- Different HTML elements
    - `headings` (e.g. `<h1>`)
    - `div`s (`<div>`)
    - `paragraphs` (`<p>`)
    - `button`s (`<button>`)
    - `section` (`section`)
    - `image` (`<img>`)

- Element attributes such `class` & `id`

- I learned the uses of each of the HTML elements & a few attributes

- Adding <img> elements and linking them to either files on your device or online.

#### 🧠 Key concepts to remember
<!-- The 2–5 things you absolutely must not forget. Short, punchy. -->

| Concept | One-line summary |
|---------|-----------------|
| The use of `headings`, `divs`, `paragraphs`, `buttons` | Different levels of elements are used for different things: `<h1>` for headings `<button>` for buttons |
| The `class` attribute | To classsify elements that, for instance, you would want to style in the same way |
| The `id` attribute | To uniquely class elements that you want to work on uniquely |

**Note**: Styling is not the only thing that these attributes are used for, they can also be used to class elements for scripting

#### 🔬 Code snippets / examples
<!-- Paste any small examples that clicked for you. Add a comment above each explaining why it's here. -->

This bookstore html snippet demonstrates the use of the different HTML elements and attributes

```html
<html lang="en">
<head>
  <meta charset="UTF-8" />
  <title>XYZ Bookstore Page</title>
</head>

<body>
  <h1>XYZ Bookstore</h1>
  <p>Browse our collection of amazing books!</p>
  
  <div class="card-container">
    <div class="card" id="sally-adventure-book">
      <h2>Sally's SciFi Adventure</h2>
      <p>This is an epic story of Sally and her dog Rex as they navigate through other worlds.</p>
      <button class="btn">Buy Now</button>
    </div>
    
    <div class="card" id="dave-cooking-book">
      <h2>Dave's Cooking Adventure</h2>
      <p>This is the story of Dave as he learns to cook everything from pancakes to pasta, one recipe at a time.</p>
      <button class="btn">Buy Now</button>
    </div>
  </div>
  
  <p>Review your selections and continue to checkout.</p>
  
  <div class="btn-container">
    
  </div>
</body>

</html>
```
